🛡️

物理隔绝与生命周期 (S3 TTL)

用户上传的 PDF/Word 专利文件绝不留存在应用服务器磁盘中。前端获取 Presigned URL 直接直传云端对象存储桶，存储桶设置 24 小时硬销毁生命周期 (TTL)，到期无情物理粉碎，实现绝对的数据阅后即焚。

🔍

PII 实体脱敏拦截切面

在所有向外部 LLM 节点（Lawyer, Examiner, Judge）发送 Prompt 前，系统的代理拦截器会以正则加小模型自动对专利中的公司名称、联系方式、发明人进行遮罩脱敏。最终生成的 PDF 报告落地前再动态还原，保障核心隐私。

⏱️

3分钟优雅降级 (Seamless Handover)

在听证会因技术极为前沿导致辩论胶着时，一旦达到 3 分钟，前端 SSE 链接优雅切断，展示“合议中”提示；后端 Worker 保持不降级深度，继续全量推理，最终通过邮件等异步渠道分发报告。

🚀

智能流量金丝雀发布

Prompt 的升级伴随极高风险。网关仅会将 5% 的低危流量（如免费测试账号，或者非核心 IPC 分类的专利）引流至升级了新 Prompt 的新版 Worker 节点，监测法官宣判状态，成功后再逐渐推全。

⚡

影子测试与兼容 CI

在新版 Worker 投产前，CI 流水线会自动拉取线上历史真实的听证快照，在影子环境回放测试，若判定结果与老版偏差率 > 15%，触发编译卡点。同时强校验旧版挂起的 HITL 快照是否兼容新版本反序列化。

📊

极致 Agent 观测 (Langfuse)

引入 Langfuse 专用大模型可观测底座。精确记录并展现每一轮对抗的 Prompt 拓扑、TTFT（首字响应）、TPS（输出速率）。一旦某节点出错，抛出包括 RAG 参数、提示词的 Error Snapshot 快照，供开发人员一键重放 Debug。
