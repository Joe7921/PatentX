## 2. 详细设计与实现方案

### 2.1. 3D 专利特征星图投影 (Canvas 2D)
- **位置**: 位于授权预估概率卡片与裁决意见卡片所在网格行的右侧。扩展原 3 列网格为 `grid-cols-1 md:grid-cols-3 lg:grid-cols-4`：
  - 授权预估概率 (col-span-1)
  - 裁决意见卡片 (lg:col-span-2)
  - 3D 专利星图 (col-span-1 md:col-span-3 lg:col-span-1)
- **数据绑定**:
  - 中心渲染国内专利核心（黄色/蓝色发光球体，随时间有发光波纹的呼吸效果）。
  - 周围轨道上分布着 `patents` 中的现有技术（行星球体，例如 `EP3812049A1`）。
  - 对每个行星，如果在 `matrices[pat.id]` 中包含 any 一项的状态为 `Fully_Disclosed` 或 `Conflict`，则判定该行星为“冲突特征行星”，渲染为醒目的红色/粉色球体，并在每一帧朝外释放暗红尘埃粒子（抛射运动）。如果没有冲突，则为青色/蓝色球体。
- **数学实现**:
  - **公转**: 行星在各自轨道上以角速度 $\omega_i$ 绕 Y 轴公转。
  - **视点旋转**: 支持鼠标在 Canvas 拖拽。监听 `mousedown`/`mousemove`/`mouseup`，计算鼠标偏移并将其累加到角速度上，每一帧对世界空间坐标 `(x_w, y_w, z_w)` 应用绕 Y 轴和绕 X 轴的 3D 旋转以产生 `(x_r, y_r, z_r)`，角速度带阻尼衰减（0.94）。
  - **透视投影**: 
    $x_{2d} = centerX + x_r \cdot \frac{focalLength}{z_r + zOffset}$
    $y_{2d} = centerY + y_r \cdot \frac{focalLength}{z_r + zOffset}$
    其中 $zOffset$ 确保行星在视口前方。
  - **景深感**: 绘制半径随 $z_r$ 缩放 $R_{2d} = R_{base} \cdot \frac{focalLength}{z_r + zOffset}$，并根据 $z_r$ 调节 alpha 透明度与高亮。
  - **流光数据粒子**: 在中心与各行星的连线上绘制粒子。粒子在 3D 连线段上以比例 $t_{flow}$ 从中心滑向行星，同样应用 3D 旋转和透视投影绘制，呈现逼真的 3D 动态流动。
  - **行星抛射微尘**: 冲突行星在其三维坐标处产生随机方向的微尘，并在 3D 空间中受重力和阻力作用抛射扩散，生存时间结束后消失。

### 2.2. Agent CoT 折叠思维链气泡
- **位置**: 位于左侧“多智能体嵌套辩论日志”中的每一个非系统（普通 Agent）发言气泡内部。
- **组件结构**:
  - 发言文字下方添加一个按钮“展开思维链 (CoT)”，可切换展开/折叠。
  - 展开区域采用 `framer-motion` 驱动高度和透明度平滑伸缩。
  - 内容包括：
    - **思维链流程大纲**：根据 Agent 角色（审查员、申请人、法官、通用）定制化展示三步流程大纲。
    - **资源耗费计量**：动态计算该发言估算的输入 Token、输出 Token 及预算消耗，维持高拟真度与无硬编码规则。
  - 样式：磨砂玻璃特效（`backdrop-blur-md bg-slate-950/40 border border-slate-800/80`），与主面板视觉契合。

## 3. 验证方案与结果
- **运行命令**: 在 `frontend` 目录运行 `npm run build` 确保完全无错无警告。
- **构建结果**: 验证通过，编译打包 100% 成功，输出 JS bundle 正常，未产生任何 TypeScript 类型或构建错误。
- **视觉和交互确认**:
  - Canvas 自适应宽度，鼠标拖拽可围绕 3D 空间平滑旋转，具有明显的近大远小与深度透明度变化。
  - 中心为带有蓝色/黄色脉冲波纹的发光恒星，周边为行星（对比文献 D1, D2），外层包含特定对比特征卫星（D1_1, D1_2）。
  - 当 SSE 播放辩论日志时，在被提及 the feature node 上流动亮蓝色光斑，并产生连线加粗流光效果。
  - 挂起分析时，冲突节点（Fully_Disclosed）自动变红并持续扩散红色微尘粒子。
  - 对话气泡内嵌了精美的折叠按钮，支持原地 Morphing 平滑展开，显示 3 步白盒思维链步骤与 monospace Token 消耗统计。
