# Milestone M4: 挂起暗淡与单元格批注粒子反馈

## 任务目标
实现 PatentX UI/UX 全景重塑的 Milestone 4 交互动效与功能：
1. **全局遮罩暗淡**：当收到 `hitl_interrupt` 并且系统处于 `PAUSED` 状态时，主控台除冲突特征与输入卡片外，其他干扰面板全部半透明暗淡（Dampen）。
2. **冲突单元格呼吸**：矩阵中发生新颖性冲突（`Fully_Disclosed`）的单元格（状态标签）开始做波纹呼吸发光。
3. **原地 Morphing 展开微型批注卡片**：用户直接点击该发光单元格（或该行），即在原地 Morphing 展开一个微型的批注卡片，允许专家输入修正意见。
4. **粒子注入双向反馈**：
   - 专家在单元格微型卡片中输入批注并点击“保存”时，批注化作一颗“微光能量粒子”，从批注框中以抛物线轨迹流畅地飞入并吸附在对应的对齐矩阵单元格上，单元格状态标签瞬间 Morphing 改变为亮青色的“专家已批注”标签。
   - 在底部的全局 `AgenticPauseCard` 点击“专家修正”按钮时，将收集的所有批注（若为空则自动为 4 个冲突行分配批注）从全局输入框底部射出微光粒子，抛物线飞入所有对应的单元格，吸附并改变状态为亮青色。
   - 随后，全局卡片物理收缩，面板渐变解冻亮起，调用 `resumeAnalysis` 接口并将专家批注字典发送给后端，恢复分析流。
5. **构建验证**：运行 `npm run build`，确保整个前端项目编译通过，零错误零警告。
